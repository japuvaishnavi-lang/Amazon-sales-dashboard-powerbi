#  Amazon Sales Analysis Dashboard (Power BI)

##  Project Overview
This project presents an interactive Power BI dashboard for analyzing Amazon sales data. It provides insights into sales performance, product categories, fulfillment methods, order status, and geographical distribution.

##  Objectives
- Analyze overall sales performance
- Track monthly sales trends
- Identify top-performing product categories
- Analyze fulfillment methods
- Monitor order status
- Identify top-performing states and cities
- Provide business insights

## Dashboard Features

### KPI Cards
- Total Sales
- Total Orders
- Total Quantity
- Average Order Value
- Total States

### Visualizations
-  Monthly Sales Trend
-  Sales by Category
-  Sales by Fulfilment
-  Order Status Distribution
-  Category-wise Sales (Treemap)
-  Top 10 States by Sales
-  Top 10 Cities by Sales

### Filters
- Date
- Category
- Fulfilment
- State
- Sales Channel

##  Tools Used
- Power BI Desktop
- Power Query
- DAX

##  Dataset
Amazon Sales Dataset
